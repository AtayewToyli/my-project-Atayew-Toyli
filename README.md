# Мой учебный проект

![Version 1.0.0](https://img.shields.io/badge/version-1.0.0-badge.svg)

## Description
Это программа на C, которая выводит информацию о студенте и контрольной работе. Проект создан для изучения Git.

![Логотип Git](https://git-scm.com/images/logos/downloads/Git-Logo-2Color.png)

> Проект создан для освоения Git и работы с ветками

**Задачи проекта:**
- Вывод информации о студенте
- Изучение Git веток
- Создание README документации
